# Code for querying data from MRO enclosures, cameras, scopes, and weather.
* API/enclosures: querys REST API for all three enclosures and uses ZWO to extract data from scope cameras.
* API/cameras: querys the build-in web API for the PTZ site camera as well as the three Axis eclosure cameras.
* API/weather: querys the Boltwood and Davis Weather Station's default web API to get live MRO weather data.
* bots: code for messaging bot.
